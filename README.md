# Data Analysis Pro

This is a browser-based data analysis tool that allows you to upload, analyze, and visualize data directly in your browser. No server-side processing is needed.

## Features

*   Upload Data: Upload data from CSV files or by pasting raw data.
*   Data Overview: Get a quick overview of your data, including the number of rows, columns, and missing values.
*   Visualizations: Generate various visualizations to understand your data, such as:
    *   Histograms
    *   Scatter plots
    *   Box plots
    *   Heatmaps
*   Statistical Analysis: Perform statistical analysis on your data to find insights.
*   Machine Learning: Apply machine learning models like:
    *   Linear Regression
    *   K-Means Clustering
    *   Decision Trees
*   Data Cleaning and Transformation**: Clean and transform your data with features like:
    *   Missing value imputation
    *   Feature scaling (Min-Max, Standardization)
    *   Column merging and splitting
*   Export: Export your cleaned data or analysis results to JSON or CSV.

# How to Use

1.  Open the `index.html` file in your web browser.
2.  Upload a CSV file or paste your data into the provided text area.
3.  Use the navigation menu on the left to select the analysis, visualization, or transformation you want to perform.

# Technologies Used

*   HTML5
*   CSS3
*   JavaScript
*   [Papa Parse](https://www.papaparse.com/) - for CSV parsing
*   [Chart.js](https://www.chartjs.org/) - for creating charts
